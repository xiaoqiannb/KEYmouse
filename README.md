# 一个基于Rust的windows端不用再碰鼠标的工具
### 这个工具主打一个无头，你将不会看到任何运行后的变化直到你按下按键

## 使用方法
下载exe双击运行，就这么简单
### 如果需要关闭软件需要在任务管理器搜索exe的名字并且手动结束进程


## 配置文件使用json格式，举例的文件如下

```json
{
"modifier_key": "lalt",
"toggle_key": "grave",
"up_key": "i",
"down_key": "k",
"left_key": "j",
"right_key": "l",
"left_click_key": "lbracket",
"right_click_key": "rbracket",
"move_distance": 5
}
```
|配置项| 功能               |
|----|------------------|
| `modifier_key` | 修饰键，用于触发其他键的组合   |
| `toggle_key` | 切换键，用于切换开关模式     |
| `up_key` | 上键               |
| `down_key` | 下键               |
| `left_key` | 左键               |
| `right_key` | 右键               |
| `left_click_key` | 左键点击键            |
| `right_click_key` | 右键点击键            |
| `move_distance` | 移动距离，使用int值      |
